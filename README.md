## Stack
*  Node JS + Express JS
*  Express Boom (Http Error Message)
*  Sequelize (ORM)
*  PostgreSQL
*  Winston (Logger)
*  Convict (Environment)
*  Chai + Mocha + Nyc (Test + Report)
*  Eslint (Maintain code pattern)
*  Swagger (API Documentation)
